# Clone the sample project.

```shell
mkdir test-out-namicorn && cd $_
git clone https://github.com/unstoppabledomains/namicorn-searchbar.git ./
```

# Install dependencies.

```shell
yarn install
```

# Run the project in the development mode

```shell
yarn start
```
